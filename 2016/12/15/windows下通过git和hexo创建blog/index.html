Node.js安装

下载并安装Node.js　(这里没啥好讲的，默认安装就行了，安装路径可以自己设置下)

Git安装

过两天会发一篇关于如何安装GIT的文章，并讲解一些GIT语句和基本操作，这里就先不叙述了。QAQ~GIT安装与基本命令

Hexo安装

鼠标右键任何地方，选择Git Bash，并使用一下命令安装：

$ npm install hexo-cli -g

如果之后在使用的过程中，遇到以下错误

ERROR Deployer not found :github

则运行一下命令

$ npm install hexo-deployer-git --save
$ hexo g
$ hexo d

然后创建存放博客Blog的文件夹，随你创建(比如我是安装在D:\hexo),进入D:\hexo文件夹，右击鼠标，选择Git Bash,执行以下命令，Hexo会自动在该文件夹下下载搭建网站所需的所有文件。

$ hexo init

安装依赖包

$ npm install

这是hexo基本安装完成了，下面看看效果，执行以下命令：

$ hexo g
$ hexo d

用浏览器访问localhost：4000或127.0.0.1:4000，这是就可以看到一个漂亮的博客blog了。有时4000端口可能会被占用，这是调用一下命令：

$ hexo server -p 5000

5000只是一个例子，自己也可以随便调用6000等等其他端口。当然此时搭建的博客是本地的，想要将它放置到github上，这时就需要下面的步骤了。

注册Github账号

作为一个程序员，Github账号总要有一个吧。Github

创建Repository

repository相当于一个仓库，用来存放自己的代码文件。一个项目对应一个repository,点击+并选择New repository,

进行如下图类似的填写,Repository name填写为yourname.github.io(yourname填写为github的username(没试过其他的行不行，自己可以试下)),点击Create repository，创建repository成功。

部署hexo文件到Github(需要配置ssh)

打开D:\hexo，并打开_config.yml,翻到最后，如下：

deploy:
  type: git

将之配置为(这种配置是不需要http://)：

deploy:
  type: git
  repository: git@github.com:HelloWorldhao/HelloWorldhao.github.io.git
  branch: master

配置完_config.yml后(上述是我的repository)，需要配置Git的SSH key，步骤如下：
鼠标右击任意位置，选择Git Bash，

$ ssh-keygen -t rsa -C "your_email@youremail.com"

后面的your_email@youremail.com改为自己的邮箱,之后会要求确认路径和输入密码，选择默认一路回车就行。最后如果SSH成功生成，就会在C:\Users\Administrator下生成.ssh文件夹,进入文件夹，可以看到有三个文件id_rsa,id_rsa.pub,known_hosts。打开id_rsa.pub,全选,复制,打开github,进入Settings,可以看到Profile下有一个SSH and GPG keys，点击进入，点击New SSH key，title可以任意填写，将刚刚copy的粘贴到Key,点击Add SSH key，创建成功。验证SSH key创建成功,在Git Bash下输入：

$ ssh -T git@github.com

如果是第一次会提示是否continue,输入yes可以看到：You’ve successfully authenticated,but GitHub does not provide shell access,这就表示已经成功连接上Github了。
上传文件之前需要设置username和email，这是github用来记录上传者的信息。

$ git config --global user.name "your name"
$ git config --global user.email "your_email@youremail.com"

最后就是上传文件了。

$ hexo g
$ hexo d

在浏览器中输入yourname.github.io,就可以看到部署上去的博客了。(yourname改为自己填写的username)
